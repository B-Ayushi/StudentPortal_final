How This Portal Works

Each component demonstrates a real cloud concept

👤

User Auth

Register and login with hashed passwords. JWT tokens secure every API call for protected actions like project submission and file management.

📝

Project Submission

Structured metadata like title, description, and tech stack is stored in a managed cloud database using Supabase PostgreSQL.

📤

File Upload

Files such as PDFs, ZIPs, and images are stored in Supabase Storage. Metadata remains in the database for easy lookup and tracking.

🔌

REST API

All actions are handled through secure backend API endpoints for authentication, submissions, uploads, listing, downloads, and deletions.

☁️

Cloud Deployment

The full-stack app runs in the cloud with a hosted backend and managed services, making it accessible from any device with a browser.

📊

Scalable Architecture

The app separates frontend, backend, database, and storage concerns so it can grow cleanly as more users and files are added.
